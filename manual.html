ゲームに戻る
説明書
ニッポンつなぎとは

日本全国の市区町村名を使ったしりとりゲームです。コンピューターが最初の地名を提示するので、その最後の文字から始まる市区町村名を答えてください。

連続正解でコンボボーナスが加算され、スコアがどんどん伸びていきます。どこまで続けられるか挑戦してみてください！

ルール
  • コンピューターが出した市区町村名の、最後のひらがなから始まる市区町村名を入力してください。
  • 「市・区・町・村」はしりとりの対象外です。たとえば「横浜市」なら「よこはま」の最後の文字「ま」から始まる地名を答えます。
  • 濁点・半濁点の有無は問いません。「が」と「か」、「ぱ」と「は」などは同じ文字として扱います。
  • すでに答えた地名は使えません。
  • 3回ミスすると候補ヒントが表示されます。
  • 続けられなくなったらギブアップボタンでゲームを終了できます。
スコアシステム

正解するたびに 1 ＋ コンボ数 × 0.2 点が加算されます。コンボを重ねるほど1問あたりの得点が上がります。

入力した地名の記録

過去のプレイで正解した地名はすべて端末に保存されます。スタート画面の「入力した地名を見る」ボタンから、答えた地名を五十音順で確認できます。

全国約1,885件の市区町村のうち、何件制覇できるか挑戦してみてください。

BGMについて

BGMはブラウザ標準の Web Audio API を使ったオリジナル楽曲です。都節音階（みやこぶしおんかい）をベースにした和風のオルゴール調メロディーです。

音量調節・ON/OFFの設定はスタート画面上部のコントロールから行えます。設定は次回起動時にも引き継がれます。
